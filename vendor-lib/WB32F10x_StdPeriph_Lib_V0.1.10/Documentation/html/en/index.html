WB32F10x Standard Peripherals Library

The WB32F10x standard peripheral firmware library is a series of functions, data structures and macro definitions prepared for the characteristics of the WB32F10x peripherals. The function library also contains the driver description and application examples of each peripheral. By using this firmware function library, there is no need to master the details deeply, Users can also easily apply each peripheral. Therefore, the use of this firmware function library can greatly reduce the user's program development time, thereby reducing development costs.

Each peripheral driver consists of a set of functions, this set of functions covers all the functions of the peripheral. The development of each device is driven by a common API, which standardizes the structure, function and parameter names of the driver.

Because the firmware library is universal and includes the functions of all peripherals, the size and execution speed of the application code may not be optimal. For most applications, users can use it directly. For those applications that have strict requirements on code size and execution speed, The firmware library driver can be used as a reference material for how to set up peripherals and adjust it according to actual needs.
