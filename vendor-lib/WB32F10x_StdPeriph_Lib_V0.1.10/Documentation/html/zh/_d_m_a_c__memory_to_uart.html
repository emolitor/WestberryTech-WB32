例程描述

本例程演示如何使用DMAC将数据从Memory传输到UART。

本例程系统时钟工作在96Mhz，通过PLL配置产生。PLL时钟源是外部8Mhz晶振。APB1的分频比是1，APB2的分频比是1。

在本例程中，memSrc作为DMA传输的源，UART2->THR作为DMA传输的目标。 当本例程运行后，会通过UART1打印出相关信息，输入任意字符启动DMA传输。 当DMA传输完成后，UART2会发出"Hello world!!!"。

  • 硬件平台搭建
    • HSE 连接外部8MHz晶振
    • UART1 (PA9, PA10) 连接到PC
    • UART2 TX (PA2) 连接到PC
如何使用？

为了使程序工作，您必须执行以下操作 :

  • 使用 MDK5 打开此例程文件夹中的 DMAC_MemoryToUart.uvprojx 工程
  • 重新编译该工程并下载到目标存储器中

  • 运行此例程
