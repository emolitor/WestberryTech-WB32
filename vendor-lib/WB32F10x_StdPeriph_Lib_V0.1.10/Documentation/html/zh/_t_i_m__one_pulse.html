例程描述

这个例子说明了如何使用TIM外围设备在定时器输入引脚中接收到外部信号的上升沿后生成单脉冲模式。
系统时钟工作在96Mhz，通过PLL配置产生。PLL时钟源是外部8Mhz晶振。APB1的分频比是1，APB2的分频比是1。
TIM1CLK = SystemCoreClock：
-预分频器 = 0
自动重载值为65535（TIM1->ARR），所以触发TIM1输入的最大频率值是96000000 / 65535 = 1464Hz。
TIM1配置如下：
使用单脉冲模式，外部信号连接至TIM1_CH2引脚（PA.09）。
上升沿作为有源边，输出单脉冲信号
在TIM1_CH1（PA.08）上。
Tim_脉冲定义延迟值，延迟值固定为：
delay = CCR1 / TIM1计数器时钟
= 16383 / 96000000 = 170.65 us。
（TIM_period - TIM_pulse）定义一个脉冲值，脉冲值固定为：
一个脉冲值 =（Tim_周期 - Tim_脉冲）/ Tim1 计数器时钟
=（65535 - 16383） / 96000000 = 512us。
