例程描述

这个例子展示了如何配置TIM1外围设备来生成一个互补的TIM1信号，插入一个定义的死区时间值，使用中断特性并锁定所需的参数。
系统时钟工作在96Mhz，通过PLL配置产生。PLL时钟源是外部8Mhz晶振。APB1的分频比是1，APB2的分频比是1。
TIM1CLK为96MHz，TIM1预分频器等于0，因此使用的TIM1计数器时钟为96MHz。
TIM1 频率 = TIM1 counter clock / (TIM1_Period + 1) = 1.46KHz。
TIM1 Channel1 占空比 = TIM1_CCR1 / (TIM1_Period + 1) = 50。
TIM1 Channel1N 占空比 = (TIM1_Period - TIM1_CCR1) / (TIM1_Period + 1) = 50。
在不同的互补信号之间插入100 / 96Mhz 的死区时间，选择锁定级别1。
Break Polarity 设为高。
使用示波器可以显示TIM1波形。

硬件和软件环境
  • 硬件平台搭建
    • HSE 连接外部8MHz晶振
    • 将TIM1引脚连接到示波器上，以监测不同的波形:
      • TIM1_CH1 pin (PA.08)
      • TIM1_CH1N pin (PB.13)
    • 连接 TIM1 break pin TIM1_BKIN pin (PA.06) to the VCC 来生成Break。
